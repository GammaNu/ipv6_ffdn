# Usage

Installation:

    git clone https://github.com/Psycojoker/ipv6_ffdn

    cd ipv6_ffdn

    virtualenv ve
    ve/bin/pip install -r requirements.txt

Run it:

    ve/bin/python ipv6_ffdn.py
